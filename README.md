# FinalProjectSkeletonCode-2025Fall

The main project repository contains 2 sub-folders:
- CodeWithoutFirebase 
- CodeWithFirebase

You are recommended to start with the CodeWithoutFirebase folder first test and tune the basic functionalities of your smart car. Once everything is functioning well, you can proceed to the CodeWithFirebase folder (replace those files you have edited) to integrate Firebase functionalities of getting realtime examStates and traffic light status.

## Arduino Libraries to Install:
(For CodeWithFirebase)
- FirebaseClient (version: >= latest 2.0.1)
- ArduinoJson (version: >= latest 7.3.1)
